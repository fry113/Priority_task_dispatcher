# Priority Task Dispatcher

Библиотека и демонстрационное приложение на C++23 для диспетчеризации задач с приоритетами.

## Что делает проект

- Принимает задачи с разными приоритетами.
- Складывает их в приоритетную очередь.
- Выполняет задачи в пуле потоков.
- Поддерживает гибкую конфигурацию очередей для каждого приоритета.

По умолчанию доступны два приоритета:

- High
- Normal

Для High используется ограниченная очередь, для Normal - неограниченная.

## Основные компоненты

- TaskDispatcher - публичный интерфейс для планирования задач.
- PriorityQueue - координирует очереди по приоритетам.
- ThreadPool - исполняет задачи в рабочих потоках.

## Быстрый запуск

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
./build/dispatcher
```

## Тесты

```bash
ctest --test-dir build --output-on-failure
```

Также можно запустить тестовый бинарник напрямую:

```bash
./build/tests/dispatcher_test
```

## Структура

- include/ - заголовочные файлы.
- src/ - реализация библиотеки.
- tests/ - модульные тесты на GoogleTest.
- main.cpp - пример использования диспетчера.
